# Python Quests – Learn Python by Coding

## Overview

This repository contains solutions to the **Python Quests assignment**, a series of programming challenges designed to build a strong foundation in Python programming.

The quests progress through multiple levels, starting from basic concepts like printing and variables, and gradually advancing to loops, functions, and small programming challenges.

The goal of this project is to practice problem-solving, logical thinking, and core programming skills by implementing Python scripts for each quest.

---

## Project Structure

```
group3-python-quests/

├── quests/
│   ├── quest_01_first_spell.py

│   ├── ...

│
└── README.md
```

Each quest is implemented as a separate Python script inside the **quests/** directory.

---

## Learning Objectives

This project covers the following Python fundamentals:

* Printing output with `print()`
* Variables and data types
* User input using `input()`
* Type conversion
* Arithmetic operations
* Conditional statements (`if`, `elif`, `else`)
* Logical operators
* Loops (`for` and `while`)
* Functions and reusable code
* Basic problem solving with Python

---

## Levels and Quests

The quests are divided into six progressive levels:

**Level 1 – The Magic of Words and Numbers**

* Introduction to printing, variables, and data types.

**Level 2 – A Dialogue with the Machine**

* User input and basic arithmetic operations.

**Level 3 – The Crossroads of Logic**

* Conditional statements and decision making.

**Level 4 – The Power of Repetition**

* Loops and repeated execution of code.

**Level 5 – The Alchemist's Lab**

* Functions, parameters, and returning values.

**Level 6 – The Grand Challenge**

* Applying all learned concepts to solve more complex problems.

---

## How to Run the Scripts

1. Ensure Python is installed on your system.

```
python --version
```

or

```
python3 --version
```

2. Navigate to the repository folder.

3. Run any quest script using:

```
python3 quests/quest_01_first_spell.py
```
